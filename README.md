# PTSNS NIRF Webpage

Pandit Shambhunath Shukla University NIRF Data and Rankings webpage.

## Project info

Built with:
- Vite
- React
- TypeScript
- Tailwind CSS
- shadcn/ui

## Getting Started

1. Install dependencies:
   ```sh
   npm install
   ```

2. Run development server:
   ```sh
   npm run dev
   ```

3. Build for production:
   ```sh
   npm run build
   ```
